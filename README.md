## Table Booking 

A fully custom-designed and unique restaurant table booking app, built entirely from scratch with original UI/UX design and concept by the developer.
Users can explore restaurant categories, browse menus, book tables, leave reviews, and locate the restaurant on Google Maps.

This project is unique: every screen, interaction, and design choice was crafted and ideated personally, showcasing original creative and technical skills.

Live Demo: https://table-booking-qs03.onrender.com

# 🛠️ Tech Stack
- React
- Vite 
- Tailwind CSS
- React Router
- Context + useReducer 
- React Hook Form 
- Sooner 
- NavLink for navigation
- Lazy()
- ESlint
- JavaScript (ES6+)

#🚀 Features
- 🏷 Restaurant categories on homepage  
- 📋 Menu browsing with food details & prices  
- 🗓 Table booking with **availability check** (prevents double-booking for same time & date)  
- ⭐ Leave reviews & feedback for restaurant
- ⏳ Loading & error handling
- 📍 Google Maps integration to find restaurant location  
- 🌗 Day/Night toggle mode  
- ⚡ Smooth animations using animation libraries  
- 📝 Responsive and user-friendly UI/UX
